    Wi-Fi

    Wi-Fi is everywhere‚Äîhomes, schools, caf√©s, offices‚Äîbut many beginners use it every day without really understanding how it works. This tutorial explains what Wi-Fi is, how it works, and how to use it properly, step by step.

    1. What Is Wi-Fi?

    Wi-Fi is a technology that allows devices to connect to a local network wirelessly using radio waves.

    With Wi-Fi, your devices can:

    • Access the Internet
    • Communicate with other devices on the same network
    • Avoid physical cables (Ethernet)

    üìå In simple terms:

    Wi-Fi = wireless version of Ethernet

    2. Wi-Fi vs Internet (Very Important!)

    Many beginners confuse these two concepts.

    Term Meaning
    Wi-Fi A wireless connection between your device and a local network
    Internet The global network of networks

    üí° You can have:

    • Wi-Fi without Internet (local network only)
    • Internet without Wi-Fi (wired connection)

    Example:

    • Your laptop connects to a router via Wi-Fi
    • The router connects to the Internet via fiber or cable

    3. Basic Wi-Fi Components

    A typical Wi-Fi setup includes:

    1Ô∏è‚É£ Router (Core Device)

    • Sends and receives Wi-Fi signals
    • Connects your local network to the Internet
    • Usually called a Wi-Fi router

    2Ô∏è‚É£ Access Point (AP)

    • A device that provides Wi-Fi coverage
    • Home routers usually include an AP
    • In offices, APs may be separate devices

    3Ô∏è‚É£ Client Devices

    • Phones
    • Laptops
    • Tablets
    • Smart TVs
    • IoT devices

    4. How Wi-Fi Works (High Level)

    1. The router creates a wireless network
    2. Your device searches for available Wi-Fi networks
    3. You select a network (SSID) and enter a password
    4. Data is transmitted via radio waves
    5. The router forwards traffic to the Internet

    üì° Wi-Fi uses radio frequencies, not cables.

    5. What Is an SSID?

    SSID = Service Set Identifier

    It is simply the Wi-Fi network name you see.

    Examples:

    • Home_WiFi
    • CoffeeShop_Free
    • GL-MT3000

    üìå Multiple routers can broadcast different SSIDs in the same area.

    6. Wi-Fi Frequency Bands

    Wi-Fi works on different frequency bands:

    2.4 GHz

    • Longer range
    • Slower speed
    • More interference

    5 GHz

    • Shorter range
    • Faster speed
    • Less interference

    6 GHz (Wi-Fi 6E / 7)

    • Very fast
    • Very clean spectrum
    • Short range
    • Requires newer devices
    Band Range Speed Interference
    2.4 GHz Long Low High
    5 GHz Medium High Medium
    6 GHz Short Very High Low

    7. Wi-Fi Standards (802.11)

    Wi-Fi versions are defined by IEEE 802.11 standards.

    Marketing Name Standard Typical Speed
    Wi-Fi 4 802.11n Up to ~600 Mbps
    Wi-Fi 5 802.11ac Up to ~3.5 Gbps
    Wi-Fi 6 802.11ax Faster + lower latency
    Wi-Fi 6E 802.11ax (6 GHz) Even faster
    Wi-Fi 7 802.11be Ultra-high speed

    üìå Newer Wi-Fi = better speed, efficiency, and stability.

    Common Security Types

    Type Status

    9. Wi-Fi Password vs Router Password

    These are different:

    Password Type Purpose
    Wi-Fi password Connect devices to Wi-Fi
    Router admin password Configure the router

    ‚ö†Ô∏è Never leave the router admin password as default.

    10. Common Wi-Fi Problems (and Why They Happen)

    Slow Wi-Fi

    • Too many devices
    • Weak signal
    • Interference
    • Old router

    Weak Signal

    • Walls and floors block signals
    • Router placed too far away

    Frequent Disconnections

    • Overlapping channels
    • Power-saving settings
    • Poor firmware

    11. Basic Wi-Fi Optimization Tips

    ‚úî Place router in a central location

    ‚úî Use 5 GHz when close to the router

    ‚úî Update router firmware

    ‚úî Change default SSID and password

    ‚úî Avoid placing router near microwaves or metal objects

    Feature Wi-Fi Ethernet
